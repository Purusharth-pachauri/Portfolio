# Portfolio - Purusharth Pachauri

Personal portfolio website for Purusharth Pachauri, showcasing skills, experience, and projects in Cyber Security, Machine Learning, and Software Engineering.

## Features
- **Projects Section:** Highlights key projects such as the Network Intrusion Detection System (ML-based NIDS), AES File Encryption & Decryption, and Image Steganography.
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Clean UI:** Built with HTML, CSS, and Vanilla JavaScript.

## Setup
To run the portfolio locally, simply clone the repository and open the HTML file:
1. Clone this repository to your local machine.
2. Open `portfolio.html` in any modern web browser.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
