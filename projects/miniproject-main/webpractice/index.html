Course Notes

Study Guide

1. HTML (HyperText Markup Language)

  • Semantic Tags: <header>, <nav>, <main>, <section>, <article>, <aside>, and <footer> give meaning for Accessibility and SEO.
  • Typography: <h1>-<h6>, <p>, <strong>, <em>, <blockquote>.
  • Lists: <ul>, <ol>, <li>.
  • Images & Tables: <img> with crucial alt attribute. Tables organize data cleanly.
  • Forms & Inputs: Forms collect user input efficiently with labels and inputs.

2. CSS (Cascading Style Sheets)

  • Variables: Custom properties defined in :root for reusable styling.
  • The Box Model: Content, Padding, Border, Margin. box-sizing: border-box is essential.
  • Layouts: Flexbox for 1D (rows/cols), Grid for 2D complex layouts.
  • Pseudo-Classes: Styling states like :hover or :focus.
  • Media Queries: For responsive design across different screen sizes (mobile friendliness).

3. JavaScript

  • Variables & Types: const, let, String, Number, Array, Object.
  • DOM Manipulation: Use getElementById or querySelector to read and change HTML elements dynamically.
  • Functions: Reusable code blocks (standard and arrow functions).
  • Event Listeners: Responding to user actions like 'click' and form 'submit'. Calls preventDefault() to stop reloads.
  • Asynchronous JS: Using setTimeout() for delays and fetching external data.
